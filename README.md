# -integracao-com-bd-relacional-para-capital-de-giro
Aplicação para controle de entradas e saídas financeiras, com integração a banco de dados SQLite e geração de gráficos para análise de fluxo de caixa.

Tecnologias: Python, SQLite, Pandas, Matplotlib

Funcionalidades:
• Cadastro de receitas e despesas
• Armazenamento em banco de dados
• Geração de gráficos financeiros
• Análise temporal dos dados
